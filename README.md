# bang-style

My favorite ruby styles

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bang-style'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bang-style

## Usage

Create a `.rubocop.yml` with the following directives:

```yaml
inherit_gem:
  bang-style:
    - default.yml
```
